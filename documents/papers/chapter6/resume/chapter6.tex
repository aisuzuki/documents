{\LARGE 第 6 章 6.1 節 はじめに \\} 
{\large 2002年 7月 9日 \hspace*{5mm} 01ca0125 鈴木 藍 }
\end{center}
\paragraph{はじめに}
線形、非線形の意味を 式、現象の視点から考察し、また、
これをそれぞれの節にあてはめた。
\section{ \large 節の概要 }
非線形については
「非線形微分方程式は大抵のものが解析的に解けない」と書いてあるだけである。\\
今まで行って来た 2階常微分方程式とは別の解法が紹介されている。
ここでは、$\frac{dy}{dx}$ を $p$ と置き換えるような 変数変換を行っている。
\section{ \large  2 階常微分方程式の一般形  }
\begin{eqnarray}
\frac{d^2 y}{d x^2} = f \Big( y, \frac{dy}{dx}, x \Big) \eqname{6.1}
\end{eqnarray}
これを言い替えると 「$y$ を $x$ で 2 階微分したものは、関数 $f$ に$ y, \frac{dy}{dx}, x$ を
与えたもの。」 
\subsection{ 例を解く  }
$\frac{dy}{dx}$ を $p$ と置き換えて計算する。(計算過程は添付資料)
すると、
\begin{eqnarray}
y &=& \pm \frac{1}{3} (2x + 2A)^{\frac{3}{2}} + B  
\end{eqnarray}
が得られる。

\section{ \large 言葉の整理  }
%------------------------------------------------------
線形、非線形 ともに どのような事をあらわしているのかをモデルと
数式から考える。
\subsection{ \normalsize 線形とは }
\begin{itemize}
\item 定数倍をするという演算と順序を交換してもよい
\item 和をとるという演算と順序を交換してもよい
\item 解の重ね合わせが可能な性質を持つ「方程式」
\end{itemize}
\subsection{ \normalsize 非線形とは }
非線形ならば、線形が成り立たないような式。
\subsection{ \normalsize 現象から考えた非線形 }
互いに作用されないのが 線形ならば、非線形は 何かに影響を
受けるようなモデルが 非線形で、
電子や、惑星も お互いに作用しあうような モデルである。 \\
$\to$ これらは、周辺との相互作用を繰り返して現象が発展してゆき、このような現象は
線形現象として記述することは出来ない。

%-------------------------------------------
\section{ \large それぞれの章の微分方程式 }
%-------------------------------------------
解法 $\to$ 惑星の運動以外は、変数変換した後に 変数分離に持ち込み、普通に積分して解いている。
式  $\to$ 線形の条件を満たしていない。

\paragraph{まとめ・結論}
世の中の現象は、全てが非線形であるので、現実の
正確なモデルを作るためにはこの非線形微分方程式を解く事が重要である。

\paragraph{参考資料}

\section{ \large 例を解く  }
例題
\begin{eqnarray*}
\frac{dy}{dx} \cdot \frac{d^2 y}{d x^2} = 1
\end{eqnarray*}
を解く。ここでも、$ \frac{dy}{dx} = p $ と置いて計算をする。
\begin{eqnarray*}
p \frac{dp}{dx} &=& 1 \cr
\int p dx &=& \int dx \cr
\frac{1}{2} p^2 &=& x + A \cr
p^2 &=& 2x + 2A    \cr
p &=& \pm (2x + 2A)^{\frac{1}{2}}
\end{eqnarray*}
$p$ を戻して、
\begin{eqnarray*}
\frac{dy}{dx} &=& \pm (2x + 2A)^{\frac{1}{2}} \cr
\frac{dy}{dx} &=& \pm \int (2x + 2A)^{\frac{1}{2}} dx 
\end{eqnarray*}
ここで、もう一度積分するが、置換積分したいので
$(2x + 2A) = P$ として$dx$、を求める。
両辺を $dx$ で微分して、
\begin{eqnarray*}
(2x + 2A) &=& P \cr
\frac{d (2x + 2A)}{dx} &=& \frac{d P}{dx} \cr
2 &=& \frac{d P}{dx} \cr
dx &=& \frac{1}{2} dP 
\end{eqnarray*}
これを $dx$ にあてはめて、
\begin{eqnarray*}
\frac{dy}{dx} &=& \pm \int P^{\frac{1}{2}} \frac{1}{2} dP \cr
\frac{dy}{dx} &=& \pm \frac{1}{2} \int P^{\frac{1}{2}} dP \cr
y &=& \pm \frac{1}{2} \frac{(2x + 2A)^{\frac{3}{2}}}{\frac{1}{2}} + B \cr
y &=& \pm \frac{1}{2} \frac{(2x + 2A)^{\frac{3}{2}}}{\frac{3}{2}} +B \cr
y &=& \pm \frac{1}{2} \frac{2}{3} (2x + 2A)^{\frac{3}{2}} + B \cr
y &=& \pm \frac{1}{3} (2x + 2A)^{\frac{3}{2}} + B
\end{eqnarray*}
よって、解は
\begin{eqnarray*}
y &=& \pm \frac{1}{3} (2x + 2A)^{\frac{3}{2}} + B
\end{eqnarray*}
となる。
