\begin{abstract} 
\addcontentsline{toc}{section}{概要}
線形、非線形の意味を 式、現象の視点から考察し、また、
これをそれぞれの節にあてはめた。
はじめににでて来る式を変形し、どのような便利さのある式なのかを
考えた。

\end{abstract}

\addcontentsline{toc}{section}{レポートの目的}

\section{ 言葉の整理  }
%------------------------------------------------------
線形、非線形 ともに どのような事をあらわしているのかをモデルからと
数式から考える。
\subsection{ 線形とは }
\subsubsection{ 式から考えた線形 }
ある演算において
\begin{itemize}
\item 定数倍をするという演算と順序を交換してもよい
\item 和をとるという演算と順序を交換してもよい
\end{itemize}
の二つの性質を満たすとき、その「演算」は線形である。\\
ある方程式において
\begin{itemize}
\item いくつかの解を加え合わせたものがまた解になっているような、
解の重ね合わせが可能な性質を持つ「方程式」
\end{itemize}
を線形であるという。解の重ね合わせとは、グラフで言うと複数の
波が重なる時、ある点の変位はそれらの波が「それぞれ単独に」作用するときの
変位の和になる、という原理の事をいう。\\
<図形的な解釈>\\
もっとも簡単な演算を考える。正比例の関係の式
\begin{eqnarray*}
f(y) = Ay
\end{eqnarray*}
これは、もし、$y_1$ を $x$ とするならば $Ay$ は $A y_1$ である。
線形演算である $y_1$ を$C$ 倍すれば、答えも $C(A y_1)$ となる。
また、$y_1 + y_2$ を $x$ としたときも、答えは $A (y_1+y_2)$と 
なっている。図で表すと
\begin{center}
\includegraphics{liner.eps} \\ fig 1. 線形演算 $f(y) = Ay$
\end{center}
となる。
\subsubsection{ 現象から考えた線形 }
式でいう線形をまとめたが、今度は モデル化する対象となる
現象で線形とはどういう事かを考えた。\\
< 線形な現象とは >\\
\begin{itemize}
\item 一定な変化をする現象(入力を定数倍すれば、出力も定数倍) 
\item 何かに 影響を受けないような現象(重ね合わせの原理より)
\end{itemize}
このような現象は、どのようなものか。例えば今まで扱って来た
もので、フックの法則は「ばねの張力はその伸びに比例する」とあるが
これも線形と考えられる。しかし、考えられるだけで、実際はもっとなんらかの
影響を外から受けているはずである。線形な現象とは、現実にありうるのだろうか。
モデルとして簡略化した時に、線形を使うのではないかと思う。

\subsection{ 非線形とは }
\subsubsection{ 式から考えた非線形 }
非線形ならば、線形が成り立たないような式、すなわち
ある演算において
\begin{itemize}
\item 定数倍をするという演算と順序を交換してもよい
\item 和をとるという演算と順序を交換してもよい
\end{itemize}
の二つの性質を満たす事が出来ない、また、
ある方程式において
\begin{itemize}
\item いくつかの解を加え合わせたものがまた解になっているような、
解の重ね合わせが可能でない性質を持つ「方程式」
\end{itemize}
が、非線形であるということになる。ということは、重ね合わせの原理も成り立たつ
ことがない。\\
$\frac{d^2 y}{dx^2} = -y$ は、二つの解 $\sin x, \cos x$ を持つが、二つ足し合わせた 
$\sin x + \cos x$ も、また $(\sin x + \cos x)'' = -(\sin x + \cos x)$ となる。こういった
現象も成り立たないような式が、非線形である。

<図形的な解釈>\\
先程の、線形の例で挙げた
\begin{eqnarray*}
f(y) = Ay
\end{eqnarray*}
を、
\begin{eqnarray*}
f(y) = y^2
\end{eqnarray*}
と考えてみる。もし、$y = (y_1 + y_2)$ とした時、値は $y_1, y_2$ をそれぞれ
二乗したものの和の他に、余分な $2 y_1 y_2$ も足し合わされている。線形の
条件を満たしていない。
これを図で表すと
\begin{center}
\includegraphics{noliner.eps} \\ fig 2. 非線形演算 $f(y) = y^2$
\end{center}
となる。
また、これを微分方程式で考えてみる。
\begin{eqnarray*}
(y^2)' - 4y = 0
\end{eqnarray*}
この式は、微分される項が二乗されている。これを解くと、
\begin{eqnarray*}
(y^2)' - 4y &=& 0 \cr
(y^2)'      &=& 4y  \cr
y'        &=& \pm 2\sqrt{y} \hspace*{5mm}(y>0) \cr
\frac{dy}{dx} &=& \pm 2\sqrt{y} \cr
\frac{1}{\sqrt{y}} \frac{dy}{dx} &=& \pm 2\sqrt{y} \frac{1}{\sqrt{y}}\cr
\frac{1}{\sqrt{y}} \frac{dy}{dx} dx &=& \pm 2 dx\cr
\frac{1}{\sqrt{y}} dy &=& \pm 2 dx
\end{eqnarray*}
両辺を積分して
\begin{eqnarray*}
\frac{1}{\sqrt{y}} dy &=& \pm 2 dx \cr
y^{-\frac{1}{2}} dy   &=& \pm 2 dx \cr
\int y^{-\frac{1}{2}} dy   &=& \pm \int 2 dx \cr
\frac{y^{\frac{1}{2}}}{\frac{1}{2}} dy   &=& \pm 2 x + C \cr
2 y^{\frac{1}{2}}  &=& \pm 2 x + C \cr
2 \sqrt{y} &=& \pm 2 x + C \cr
\sqrt{y} &=& \pm x + C \cr
y &=& \pm (x + C)^2
\end{eqnarray*}
となる。
もし、一つの解が$y_1 = x^2$ で求められたとしても、もし定数倍した $y=yC_1 y_1 = C_1 x^2$ならば、
ここではある $(y^2)'$ から $-4y$ を引いた物が $0$にならなければならないのだから
\begin{eqnarray*}
y' = 2C_1x, \cr 
(y^2)' = 4C_1^2 x^2 
\end{eqnarray*}
となり、解にあてはまらない。微分方程式が 非線形であった為である。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\subsubsection{ 現象から考えた非線形 }
互いに作用されないのが 線形ならば、非線形は 何かに影響を
受けるようなモデルが 非線形ではないかと考える。
そう考えると、やはり現実世界の全てが相互作用しながら
存在しているのだから、全部 非線形になる。
電子や、惑星も お互いに作用しあうような 物体である。
これらは、周辺との相互作用を繰り返して現象が発展してゆき、このような現象は
線形現象として記述することは出来ない。

\subsection{ 解析的に解くとは  }
微分方程式を数式のままで厳密に解く事を解析的に解くという。
非線形系は，部分・要素に分解・分析し，それを再び集めて全体を構成するという，
線形解析で許される手法もとることが出来ない。ほとんどの場合は
数値計算によって解を求めることで解析が行われている。

\section{ 134頁 はじめに }
%-------------------------------------------
\subsection{ この章の概要  }
ここでは、非線形と名前が出ているにもかかわらず 非線形についての
解説は「非線形微分方程式は大抵のものが解析的に解けない」と書いてあるのみで
今まで行って来た 2階常微分方程式とは別の解法が紹介されている。
ここでは、$\frac{dy}{dx}$ を $p$ と置き換えるような 変数変換を行っている。
\subsection{ 2 階常微分方程式の一般形  }
以下の式
\begin{eqnarray}
\frac{d^2 y}{d x^2} = f \Big( y, \frac{dy}{dx}, x \Big) \eqname{6.1}
\end{eqnarray}
が2 階常微分方程式の一般形である。
これは「$y$ を $x$ で 2 階微分したものは、関数 $f$ に$ y, \frac{dy}{dx}, x$ を
与えたも。」 と言い替えられる。ここでは、この式の特殊なもの、$f$ は $y, \frac{dy}{dx}, x$ の
任意関数であるから これらの一つが無い場合を考えている。そうする事で、一階常微分方程式に
帰着\footnote[1]{いろいろな経過をたどって、ある状態や結果に落ち着くこと。}することが出来る。
例えば、$\frac{dy}{dx} = p$ として、$y$ が引数としてあらわれないならば
\begin{eqnarray*}
\frac{d p}{d x} = f(p, x) 
\end{eqnarray*}
とすることが出来る。
\subsection{ 例を解く  }
例題
\begin{eqnarray*}
\frac{dy}{dx} \cdot \frac{d^2 y}{d x^2} = 1
\end{eqnarray*}
を解く。ここでも、$ \frac{dy}{dx} = p $ と置いて計算をする。
\begin{eqnarray*}
p \frac{dp}{dx} &=& 1 \cr
\int p dx &=& \int dx \cr
\frac{1}{2} p^2 &=& x + A \cr
p^2 &=& 2x + 2A    \cr
p &=& \pm (2x + 2A)^{\frac{1}{2}}
\end{eqnarray*}
$p$ を戻して、
\begin{eqnarray*}
\frac{dy}{dx} &=& \pm (2x + 2A)^{\frac{1}{2}} \cr
\frac{dy}{dx} &=& \pm \int (2x + 2A)^{\frac{1}{2}} dx 
\end{eqnarray*}
ここで、もう一度積分するが、置換積分したいので
$(2x + 2A) = P$ として$dx$、を求める。
両辺を $dx$ で微分して、
\begin{eqnarray*}
(2x + 2A) &=& P \cr
\frac{d (2x + 2A)}{dx} &=& \frac{d P}{dx} \cr
2 &=& \frac{d P}{dx} \cr
dx &=& \frac{1}{2} dP 
\end{eqnarray*}
これを $dx$ にあてはめて、
\begin{eqnarray*}
\frac{dy}{dx} &=& \pm \int P^{\frac{1}{2}} \frac{1}{2} dP \cr
\frac{dy}{dx} &=& \pm \frac{1}{2} \int P^{\frac{1}{2}} dP \cr
y &=& \pm \frac{1}{2} \frac{(2x + 2A)^{\frac{3}{2}}}{\frac{1}{2}} + B \cr
y &=& \pm \frac{1}{2} \frac{(2x + 2A)^{\frac{3}{2}}}{\frac{3}{2}} +B \cr
y &=& \pm \frac{1}{2} \frac{2}{3} (2x + 2A)^{\frac{3}{2}} + B \cr
y &=& \pm \frac{1}{3} (2x + 2A)^{\frac{3}{2}} + B
\end{eqnarray*}
よって、解は
\begin{eqnarray*}
y &=& \pm \frac{1}{3} (2x + 2A)^{\frac{3}{2}} + B
\end{eqnarray*}
となる。

\section{ それぞれの章の式 }
%-------------------------------------------
それぞれの章は 非線形 2 階微分方程式を途中で導き出して
いるが、 6.2 節 惑星の運動 以外は はじめににそった解き方を行っている。
また、大体は変数分離に持ち込み、そのあと普通に積分している。
ただ、どれもある変数や関数が 二乗した形になっているので、非線形な
微分方程式である。なぜ、非線形微分方程式では、この解法を行うのかは
まだよくわからない。\\
どのモデルも、求めたい物などが、それぞれ何かに影響しあうようなものである。
例えば、惑星は太陽の影響を、追跡は追跡する物体にといった具合である。
式に関しても、やはり線形の条件を満たしていない。

\section*{まとめ・結論}
\addcontentsline{toc}{section}{まとめ・結論}
世の中の現象は、むしろ線形で表される事はなく
全てが非線形であるので、現実の 正確なモデルを作るためには
この非線形微分方程式を解く事が重要である。
