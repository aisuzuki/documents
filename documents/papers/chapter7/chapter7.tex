\section{ 章の概要  }

\subsection{ はじめに }
ここでは、微分方程式でなりたつ物を扱う。解法としては
解析的に解く事が困難だということを前提として話を進めていて、
相平面と言う二つの現象の関係を表す事が出来る座標を使って
その関数の解が、どのような軌道を取るかを予測する事で解を定
めている。
また、平衡点という特殊な点のごく近い所で解がどのように変化するかを
観測し、スケッチをもとに予測をする。

\subsection{ 競争種：生存闘争 }
ある同じ食物を巡って争う二つの生物をモデル化している。
ここではどちらか一方が強いという仮定をしている。

\section{ 演習 }
%-------------------------------------------
\subsection{ (i) 捕食者 - 被食者 }
捕食者 - 被食者 の相互作用の単純なモデルは
\begin{eqnarray*}
    \frac{dx}{dt} &=& x + y \cr
    \frac{dy}{dt} &=& -x + y
\end{eqnarray*}
であたえられる。
初期個体数が $x(0) = y(0) = 1000$ であるとして、未来の $x$ と $y$
を決定せよ。被食者はいつ絶滅するか。 \\
ここでは、$x$ を捕食者、$y$ を被食者としている。
\subsection{ 解 }
まず、この二つの式の形を見て、解を $x = e^{rt}$ と仮定する。
そして、$r$ は (7.8) 式
\begin{eqnarray*}
    r^2 - (a + d)r + ad - bc = 0
\end{eqnarray*}
から、
\begin{eqnarray*}
    r^2 - 2r + 2 = 0
\end{eqnarray*}
となる。判別式で見ると、
\begin{eqnarray*}
    \sqrt{-4} = 2i
\end{eqnarray*}
となるので、複素解となる。式を解くと、
\begin{eqnarray*}
   r &=& \frac{ 2 \pm  \sqrt{4 - 8 }}{2} \cr
   r &=& \frac{2}{2} \pm \frac{\sqrt{-4}}{2} \cr
   r &=& 1 \pm \frac{2i}{2}  \cr
   r &=& 1 \pm i 
\end{eqnarray*}
よって、$r$ は 
\begin{eqnarray*}
   r = 1 \pm i 
\end{eqnarray*}
となり、これを $x$ の解に当てはめると
\begin{eqnarray*}
   x &=& A_1 e{(1+i)t} + B_1 e^{(1-i)t} 
\end{eqnarray*}
となる。これはオイラーの公式に直すと、
\begin{eqnarray*}
   x &=& A_1 e{(1+i)t} + B_1 e^{(1-i)t} \cr
   x &=& A_1 e{t+it} + B_1 e^{t-it} \cr
   x &=& A_1 e{t} e^{it} + B_1 e^{t}e^{-it} \cr
   x &=& e^{t}(A_1 e^{it} + B_1 e^{-it}) \cr
   x &=& e^{t}(A_1 \cos t + A_1 i \sin t + B_1 \cos t - B_1 i \sin t) \cr
   x &=& e^{t}((A_1 + B_1) \cos t + i ( A_1 - B_1 ) \sin t ) 
\end{eqnarray*}
ここで、$A_1, B_1$ は定数なので、さらに $A, B$ と置き換えて
\begin{eqnarray*}
   x &=& e^{t}(A \cos t + B \sin t ) 
\end{eqnarray*}
となる。(i は考えない。)
さらに、$y$ を求める。
\begin{eqnarray*}
    \frac{dx}{dt} &=& x + y
\end{eqnarray*}
であるから、これを変形して、
\begin{eqnarray*}
    \frac{dx}{dt} &=& x + y \cr
    - y &=& - \frac{dx}{dt} + x \cr
      y &=&   \frac{dx}{dt} - x 
\end{eqnarray*}
この $x$ に、先程求めた式を代入して、
\begin{eqnarray*}
   y &=& (e^{t}(A \cos t + B \sin t ))' - e^{t}(A \cos t + B \sin t ) 
\end{eqnarray*}
$e^{t}(A \cos t + B \sin t )$ を $t$ で微分すると、
\begin{eqnarray*}
   (e^{t}(A \cos t + B \sin t ))' \cr
   &=& A(e^{t} \cos t)' +  B(e^{t} \sin t )' \cr
   &=& A(e^{t} \cos t - e^{t} \sin t) + B(e^{t} \sin t + e^{t} \cos t) \cr
   &=& A e^{t} \cos t - A e^{t} \sin t + B e^{t} \sin t + B e^{t} \cos t 
\end{eqnarray*}
$- e^{t} (A \cos t + B \sin t)$ を合わせて、
\begin{eqnarray*}
   A e^{t} \cos t - A e^{t} \sin t + B e^{t} \sin t + B e^{t} \cos t  \cr
   - e^{t} A \cos t - e^{t} B \sin t \cr 
   &=& - A e^{t} \sin t + B e^{t} \cos t  \cr
   &=& B e^{t} \cos t - A e^{t} \sin t  \cr
   &=& e^{t} (B \cos t - A \sin t )
\end{eqnarray*}
よって
\begin{eqnarray*}
  y = e^{t} (B \cos t - A \sin t )
\end{eqnarray*}
となる。
そして、初期値 は $1000$ なので $A = B = 1000$
である。
よって、それぞれの $x, y$ は
\begin{eqnarray*}
\left
\{
    \begin{array}{c}
    x = 1000 e^{t}(\cos t + \sin t) \cr
    y = 1000 e^{t}(\cos t - \sin t)
    \end{array}
\right.
\end{eqnarray*}
である。ここで、被食者、$y$ が 0 になる時を
調べるのであるから、$y=0$ となるような式を求める。
ここで、$e^{t}(\cos t - \sin t)$ の $(\cos t - \sin t)$ が
0 になればよいのであるが、もし $t$ が 0 になっても 0 にならないので
どちらかの関数に統一する。
\begin{eqnarray*}
a \sin \theta + b \cos \theta = \sqrt{a^2 + b^2} \sin(\theta + \alpha)
\end{eqnarray*}
ただし
\begin{eqnarray*}
\sin \alpha = \frac{b}{\sqrt{a^2 + b^2}} \cr
\cos \alpha = \frac{a}{\sqrt{a^2 + b^2}} 
\end{eqnarray*}
という公式を用いる。$\theta$は、$t$ なので、
わからないのは $\sqrt{a^2 + b^2}, \alpha $ である。$\alpha$は
$\sin \alpha = \frac{b}{\sqrt{a^2 + b^2}}$ から導き出す。
\begin{eqnarray*}
- \sin \alpha &=& - \frac{b}{\sqrt{a^2 + b^2}} \cr
            &=& - \frac{1}{\sqrt{2}} 
\end{eqnarray*}
なので、 $\sin \alpha = - \sin 45°$ となる。45°は $\frac{\pi}{4}$ であるので
$\alpha = - \frac{\pi}{4}$ となる。よって、$y$の式は
\begin{eqnarray*}
y = 1000e^{t}(-\sqrt{2} \sin (t - \frac{\pi}{4}))
\end{eqnarray*}
と書き換えられる。$y$ が 0 になるような、$t$ の値を求める。括弧の中を $t$ を
求める式に変形して、
\begin{eqnarray*}
- \sqrt{2} \sin (t - \frac{\pi}{4}) &=& 0 \cr
t - \frac{\pi}{4} &=& 0 \cr
t &=& \frac{\pi}{4} 
\end{eqnarray*}
ここで、答えでは 
$t = \big( n + \frac{1}{4} \big) \pi \hspace*{2mm}(n = 0,1,2 \hdots )$ としているが、確かに
$n$ の値を 変化させてゆくと、式全体が $y = 0$ となる。そして、どちらの$x, y$ も
初期値が$1000$ であるなら、第一象限に座標があるので、$y$ が 0 になるときの $t$ は
$\frac{\pi}{4}$ の時である。以下は 厳密解をプロットした図である。
