\section{ 線形2解微分方程式の解法 }
今回解く式は
\begin{eqnarray}
\frac{d^2 y}{d x^2} + a \frac{dy}{dx} + by = f(x)  \eqname{5.1}
\end{eqnarray}
で、$a, b$は定数である。この式の特徴を挙げると
\[
\left
\{
    \begin{array}{l}
    ・ 非同次である。(右辺が0でない) \cr
    ・ 線形である。(定数倍、和を取るという演算と順序を交換しても成り立つ)
    \end{array}
\right.
\]
である。
\subsection{ 何を求めるか }
$y$ は $x$ についての関数である。(5.1) 式から、$y$ がどんな関数かを
求める。\\
$\to y = \ldots $ の式 \\
又、ここの $f(x)$ は、$y(x)$ からなる関数である。

\subsection{ 手順 }
前回同様に、
\begin{enumerate} 
\item 式の形を見る
\item y を仮定する
\item 同次方程式の一般解を求める 
\item 非同次方程式の特解を求める 
\end{enumerate}
という形で求める。

\subsection{ 1. (5.1) 式の一般解の形 }
(5.1)式の一般解は 106頁 (5.3)式より、常に
\begin{eqnarray}
y(x) = y_c(x) + y_p(x)  \eqname{5.3}
\end{eqnarray}
という形をとる。
\begin{itemize}
\item $y(x) \ldots$ 求めたい関数
\item $y_c(x) \ldots$ (5.1) 式の方程式から、非同次項\footnote[1]{右辺。両辺を定数倍し、それをもとに戻す演算をしても、もとに戻らない原因となっている項} を除いた 同次方程式 \footnote[2]{左辺}の一般解
\item $y_p(x) \ldots$ (5.1) 式(非同次方程式) の一般解
\end{itemize}

\subsection{ 2. 同次方程式の一般解を求める }
ここで求めたい同次方程式は
\begin{eqnarray}
\frac{d^2 y}{d x^2} + a \frac{dy}{dx} +by = 0 \eqname{5.2}
\end{eqnarray}
である。

\subsubsection{ 2.1 $y$を仮定する }
ここでは、積分をせずに、$y = e^{mx}$ と仮定する。\\
\newpage
$\to$ 何故か？ \\
(5.2) 式の形を見て、
「線形かつ同次の微分方程式で、しかも係数が定数」という形であること、つまり
\[
\left
\{
    \begin{array}{l}
    ・ 線形である      \cr
    ・ 同次である      \cr
    ・ 係数が定数である
    \end{array}
\right.
\]
を満たしているならば、解を $e^{mx}$ とするとうまくゆく場合が多いと
わかっているからである。\\
※ $y$を仮定するにあたり \\\\
これからも、$e^{mx}$ を解として仮定する機会がでて来るが、指数が複素数だった時の
事も考える。ここで、公式
\begin{eqnarray}
\left
\{
    \begin{array}{c}
    e^{i \theta } = \cos \theta + i \sin \theta     \eqname{1.1} \cr
    e^{-i \theta } = \cos \theta - i \sin \theta    
    \end{array}
\right.
\end{eqnarray}
を使う。これは オイラーの公式と呼ばれている。
\subsubsection{ 2.2 一般解を求める }
$y=e^{mx}$ と仮定したので、これを (5.2) 式にあてはめて、
\begin{eqnarray*}
(e^{mx})'' + a(e^{mx})' + b e^{mx} &=& 0 \cr
m^2 e^{mx} + am e^{mx} + b e^{mx}  &=& 0 \cr
e^{mx} (m^2 + am + b ) &=& 0 
\end{eqnarray*}
となる。
全ての $x$ に対して成立しなければならないので
\begin{eqnarray}
m^2 + am + b &=& 0  \eqname{5.5}
\end{eqnarray}
となる。
ここで、この $m$ は、仮定した解 $y = e^{mx}$ の $m$ で
これがどのような値かを知りたい。(5.5) 式は 2次方程式なので、
$m$ の解は二つある。この二つの解を、それぞれ $m_1, m_2$ とする。
107 頁 (5.4)式 の通り、(5.2)式 の解は $y_c(x) = Ay_1(x) + By_2(x)$ という
形をとるので、それぞれの$y(x)$を $e^{m_1 x}, e^{m_2 x}$ とすると $y_c$ は
\begin{eqnarray}
y_c(x) = Ae^{m_1 x} + Be^{m_2 x}  \eqname{5.6}
\end{eqnarray}
これが (5.2)式の一般解である。こうなると、$m_1, m_2$の解が重要になる。
なぜなら、一般解は$m_1, m_2$ に依存した形になるからである。そこで
(5.5)から 判別式を作り、どの様な値をとるかによって 式の形を定める事が出来る。\\
(判別式は省略した。また、判別式(ii) の重解で、一つを $y_2 = x e^{2x}$ としているが
解説文は見付けたが、まだ理解しきれていない。考え方としては、$m_1$と$m_2$が極めて
近い値である事を仮定している。)

\subsection{ 2.3 特解を求める }
これは、初期値などを定められないと求める事が出来ない。

\section{ 107頁 9行目の「容易にわかる」は容易なのか }
容易さ加減がわからなかったので考えてみたが、どうやって
$\sin (x) + \cos (x)$ が出て来たのかがわからなかった。
しかし、$\frac{d^2 y}{d x^2} + y = 0$ を日本語で言うと、
求めたいのは \\
\begin{center}
「2階微分した $y$ と $y$ をたすと、0 になるような $y$」
\end{center}
である。
\subsection{ 一般解 }
\begin{center}
\includegraphics{sin_cos_loop.eps}\\
図 1. 微分と三角関数 
\end{center} 
を見て $\frac{d^2 y}{d x^2} + y = 0$ に代入すると
\begin{eqnarray*}
\underbrace{ - \sin(x) - \cos (x) }_{\frac{d^2 y}{dx}} + \underbrace{ \sin (x) + \cos (x) }_{y} = 0
\end{eqnarray*}
となるので、たしかに $y$ は $ \sin(x) + \cos(x)$ である。
\subsection{ 特解 }
もし、$y(x)$ が $x$ だとすると 
\begin{eqnarray*}
(x)'' + x = x
\end{eqnarray*}
となり、たしかに $y_p$ の一つは $x$である。

\section{ 演習 力学的振動 (やりなおし) }
(i) から、理解した 解法を使って解き直した。
\subsection{ (a) }
与えられた値は 全ての $t>0$ に対して、$F = 0, t=0 $ならば$x=a, \frac{dx}{dt} = 0$である。
\subsubsection{ 抵抗を無視した式の解 }
抵抗を無視した時の式は
\begin{eqnarray*}
\frac{d^2 x}{d t^2} + \omega^2 x = 0
\end{eqnarray*}
である。ここでも $x$ の解を $e^{mt}$ として式を変形する。
\begin{eqnarray*}
\frac{d^2 x}{d t^2} + \omega^2 x &=& 0 \cr
\frac{d^2 x}{d t^2} &=& - \omega^2 x \cr
\frac{d^2 e^{mt}}{d t^2} &=& - \omega^2 e^{mt} \cr
m^2 e^{mt} &=& - \omega^2 e^{mt} \cr
m^2 &=& - \omega^2 \cr
m &=& \pm i \sqrt{ \omega^2 }
\end{eqnarray*}
となる。解は $i \omega , -i \omega$ となり、$e^{i \omega t}, e^{- i \omega t}$、
よって、
\begin{eqnarray*}
x = A e^{i \omega t} + B e^{-i \omega t}
\end{eqnarray*}
である。初期値を与えられているので、定数 $A, B$ を求める。
$t = 0$ の時に $x = a, \frac{dx}{dt} = 0$ であるから
\begin{eqnarray*}
x(0) &=& A e^{i \omega 0} + B e^{- i \omega 0} \cr
x(0) &=& A + B \cr
x(0) &=& A + B = a
\end{eqnarray*}
\begin{eqnarray*}
x'(0) &=& A (e^{i \omega 0})' + B (e^{- i \omega 0})' \cr
x'(0) &=& A i \omega e^{i \omega 0} - B i \omega e^{- i \omega 0} \cr
x'(0) &=& A i \omega  - B i \omega \cr
x'(0) &=& i \omega (A - B) \cr
x'(0) &=& i \omega (A - B) = a
\end{eqnarray*}
これらの $A, B$ を連立して求める。すると、$A=B= \frac{a}{2}$ が求められる。
これを、先程求めた一般解に当てはめて、
\begin{eqnarray*}
x &=& \frac{a}{2} e^{i \omega t} + \frac{a}{2} e^{-i \omega t} \cr
x &=& \frac{a}{2} (e^{i \omega t} + e^{-i \omega t})
\end{eqnarray*}
となる。さらに、$e^{i \omega t}, e^{-i \omega t}$ はオイラーの公式を
使って直すと、
\begin{eqnarray*}
x &=& \frac{a}{2} (\cos \omega t + \sin \omega t + \cos \omega t - \sin \omega t) \cr
x &=& \frac{a}{2} (\cos \omega t + \cos \omega t) \cr
x &=& \frac{a}{2} 2 \cos \omega t \cr
x &=& a \cos \omega t 
\end{eqnarray*}
よって、特解は
\begin{eqnarray*}
x &=& a \cos \omega t 
\end{eqnarray*}
となる。これは、単振動と同じ式である。
\subsection{ (b) }
今度は 外力を考慮して解く。与えられた値は $F = F_0 \cos at, t=0$ならば$x = 0, \frac{dy}{dx} = 0$
である。式は
\begin{eqnarray*}
\frac{d^2 x}{d t^2} + \omega^2 x &=& F(t) \cr
\frac{d^2 x}{d t^2} + \omega^2 x &=& F_0 \cos at 
\end{eqnarray*}
となる。
ここで、仮定として $x = A \cos \alpha t$ としている。(答えに書いてあった為。理由はよくわからないが、
よくこの式を見ると、111 頁 (5.10) 式の右辺にある解に似ている。と、いう解釈をしてみた。)
\begin{eqnarray*}
(A \cos \alpha t)'' + \omega^2 \cos \alpha t &=& F_0 \cos \alpha t \cr
- A \alpha^2 \cos \alpha t + \omega^2 \cos \alpha t &=& F_0 \cos \alpha t \cr
A \cos \alpha t (- \alpha^2 + \omega^2 ) &=& F_0 \cos \alpha t \cr
\frac{ A \cos \alpha t (- \alpha^2 + \omega^2 )}{\cos \alpha t (- \alpha^2 + \omega^2 )} &=& \frac {F_0 \cos \alpha t }{\cos \alpha t (- \alpha^2 + \omega^2 )} \cr
A &=& \frac {F_0}{(- \alpha^2 + \omega^2 )} 
\end{eqnarray*}
式変形は以上だが、ここから 特解を求めている。求めたいのは $\alpha = \omega$ の時なので
$x = A \cos \omega t$ として求める。
\begin{eqnarray*}
(A \cos \omega t)'' + \omega^2 \cos \omega t &=& F_0 \cos \omega t \cr
- A \omega^2 \cos \omega t + \omega^2 \cos \omega t &=& F_0 \cos \omega t \cr
\end{eqnarray*}
この時点で、左辺が消えてしまう。(何故？)
今回は、ここで止まってしまった。

\subsection{ これからの予定 }
式変形を調査。また、方程式の解法をもうすこし調査し 出来るようにする。
また、独立、従属の意味も 部分的にしかわからないので
一つの解となる関数が 何故独立と関係があるのかを調べる。

まだ会っていないような気がする。第 5 章 5.2 節 力学的振動では
