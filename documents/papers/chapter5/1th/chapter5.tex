\section{ 演習 1.力学的振動 }
%------------------------------------------------------
5.2 節で述べた力学的振動モデルを用いて、抵抗を無視し(k=0)
a, b の時予測される運動を決定する。抵抗を無視したモデルは
以下の式になる
\begin{eqnarray*}
  \ddot{x} + \omega^2 x = F(t)
\end{eqnarray*}

\subsection{ a. }
すべての $t>0$ に対して、 $F=0, t=0$ で $x=a, \frac{dx}{dt}=0$
\subsection{ 解 }
$\ddot{x} + \omega^2 x = F(t)$ について解く。この式の一般解は
111 頁 式(5.10) の
\begin{eqnarray*}
  x = A \cos \omega t + B \sin \omega t
\end{eqnarray*}
を持つ。(決まりなのか？) この一般解に 値を定め、
$x$ を求める。
これに、あたえられた初期値を当てはめ、解くと
\begin{eqnarray*}
  x(0) &=& A \cos \omega 0 \cr
  a    &=& A \cos \omega 0 
\end{eqnarray*}

\begin{eqnarray*}
  \dot{x}(0) &=& B \sin \omega 0 \cr
  0          &=& B \cos \omega 0 
\end{eqnarray*}
なぜか$\sin \omega 0$ だけ微分された形になっている。
また、加法定理により、
\begin{eqnarray*}
  x &=& A \cos \omega t + B \sin \omega t \cr
  x &=& \alpha \cos (\omega t + \beta)
\end{eqnarray*}
となる。これを展開して
\begin{eqnarray*}
  x = \alpha \cos \omega t \cos \beta - \alpha \sin \omega t \sin \beta
\end{eqnarray*}
となる。ここで、 $\alpha \cos \beta = A$, $\alpha \sin \beta = B$ とすると
$A = a$, $B = 0$ であったから(？) これを代入して
\begin{eqnarray*}
  x = a \cos \omega t
\end{eqnarray*}
となる。

\subsubsection{ ソース }
\begin{breakbox}
  \begin{verbatim}
#include <stdio.h>
#include <math.h>

int main()
{
  double  t, dt, a, omega;
  dt    = 1.0;
  a     = 5.0;  /* 縦軸の幅 */
  omega = 50.0;
  for (t=0.0; t<100; t+=dt) {
    printf("%f %f \n", t, a*cos(omega*t));
  }
}
  \end{verbatim}
\end{breakbox}

\subsubsection{ 実行結果 }
a は 縦の振幅の幅で、$\omega$ は横の周波数という感じがする。

\subsection{ b. }
まだ理解できていない。
\begin{itemize}
\item 答えの欄に、$x = A \cos at$ とあるが、どこからでて来たのか？
\item その $x = A \cos at$ は一般解なのか？
\end{itemize}
変形に関しては、なんとかなっているように思える。

%-------------------------------------------
\section{ 理解できていない点 }
%-------------------------------------------
\subsection{ 線形2階微分方程式について }
まず、この本を見た限りではどこで積分しているのか
わからない。解法については、もう方法が決まっているのだから
それを使うということで納得するにしても、今までの
流れから解法が全く違うように感じる。「微分方程式を
解くことは積分する事」であるのに、この文面をみても
一見積分しているところが見当たらない、という事が
今回非常に手間取った。
新たに購入した本で調べたところ、計算過程のどこかで積分しているような
記述があった。以降、これについて調べたい。(ロンスキアンという
行列式を使っていた。)\\
また、何故ここでは「試行錯誤」を行っているのかが疑問である。
一般解が、突然でて来ているようにみえる。

\subsection{ これからの予定 }
試行錯誤を 何故しているかを考える。
「試行錯誤」の方法について、ある程度だけ理解した後、
「正確なテクニック」と言われる方法を調べ、理解する。

\section*{まとめ・結論}
\addcontentsline{toc}{section}{まとめ・結論}
まず、この微分方程式の解法について理解がたらない。
解が常にどのような形を取るかや、解く流れのようなものは
なんとなくわかるにしても、これを何故、このような解き方
をしているのか、最後の最後まではっきりしなかった。
あまり意味にこだわって考えない方がいいのだろうか。
