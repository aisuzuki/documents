\section{ 演習問題 }
%-------------------------------------------
\subsection{ 問題 }
3.4 節の記号で

\begin{eqnarray*}
  \frac{dh}{dt} &=& - \lambda h^{\alpha} , 0 < \alpha < 1 
\end{eqnarray*}

と仮定する。$h$ の予測される形を導け。どんな量をとれば、予測直線グラフが得られるか。
$ \alpha = \frac{1}{3}$ のとき 3.4 節の実験データに対してモデルをテストせよ。

\subsection{ 方針 }

まず、問題の意図を考える。 そして 式を解き、$\alpha = \frac{1}{3}$ を当てはめ、
定数を定めたあとに 実験データとならべてプロットしてみる。

%-------------------------------------------
\section{ 問題を解く }
%-------------------------------------------
\subsection{ 問題の意図 }
教科書 59 ページで 孔から流出する水の体積速度 $v$ の式を考えている。
ここでは 実験結果から、 $v$ が 瓶の流出孔から上の水の高さ $h$ に依存している事が
分かっている。そこで、$h$ が減少するにつれて 流水の体積速度も減少するという $v$ の
もっとも簡単な式を

\begin{eqnarray*}
  v &=& ah 
\end{eqnarray*}

と仮定している。ここで $a$ は 正の定数である。これは 仮定の式であり、他にも

\begin{eqnarray*}
  v &=& ah^n
\end{eqnarray*}

なども考えられる。この問題は、この仮定の式を $\frac{dh}{dt} = - \lambda h^{\alpha}, 0 < \alpha < 1$
としたときのモデルをテストする事を意図していると考える。

\subsection{ 式を解く }
与えられた式は

\begin{eqnarray*}
\frac{dh}{dt} &=& - \lambda h^{\alpha}, 0 < \alpha < 1
\end{eqnarray*}

である。
両辺を積分して (ここで変数分離形を使う)

\begin{eqnarray*}
\frac{dh}{dt} &=& - \lambda h^{\alpha} \cr
\frac{dh}{dt} \frac{1}{h^{\alpha}} &=& - \lambda h^{\alpha} \frac{1}{h^{\alpha}} \cr
\frac{dh}{dt} dt \frac{1}{h^{\alpha}} &=& - \lambda dt  \cr
\frac{1}{h^{\alpha}} dh &=& - \lambda dt  \cr
\int \frac{1}{h^{\alpha}} dh &=& - \int \lambda dt \cr
\int h^{- \alpha} dh &=& - \lambda t + c \cr
\frac{1}{- \alpha + 1} h^{- \alpha + 1} &=& - \lambda t + c \cr
h^{- \alpha + 1} &=& - ( - \alpha + 1)\lambda t + (- \alpha + 1) c 
\end{eqnarray*}

ここで、$(- \alpha + 1) c$ は 全て定数なので $c_1$ とまとめる。
よって、この式は

\begin{eqnarray*}
h^{- \alpha + 1} &=& - ( - \alpha + 1)\lambda t + c_1
\end{eqnarray*}

となる。

\subsection{ $\alpha$に $\frac{1}{3}$ をあてはめる }
以上の式に $\frac{1}{3}$ をあてはめると

\begin{eqnarray*}
h^{- \alpha + 1} &=& - ( - \alpha + 1)\lambda t + c_1 \cr
h^{- \frac{1}{3} + 1} &=& - ( - \frac{1}{3} + 1)\lambda t + c_1 \cr
h^{\frac{2}{3}} &=& - \frac{2}{3} \lambda t + c_1
\end{eqnarray*}

となる。

\subsection{ 定数をどうするか }
この式ででて来る $\lambda$ と $c$ は定数である。$\lambda$ は 
正の定数を 断面積で割ったもの $\lambda = \frac{a}{A}$ である。また
$c$ は 時間が $0$ の時の $h$ の初期位置である。どちらの定数も 教科書では
値を与えられていない。未定の定数が二つある方程式を導くことが
出来なかったので、ここでは 適当な値をいれてみて、実験結果とてらし合わせてみた。

\newpage

\subsection{ ソース }

定数 $c$ は、時間 $t$ を 実験データがある $6.5$ から始めるとして $11$ とした。
$h$ は 完全に あたりを付けて プロットした値と一致するまで 値を変えてみた。

\begin{breakbox}
   \begin{verbatim}
#include <stdio.h>
main()
{
  double  h, t, ti, c;
  c = 11;

  for (t=6.5; t<170; t+=0.1) {
    h = -(2.0/3.0)*(0.7/7.0)*t+c;
    printf("%lf %lf \n", t, h);
  }
}
   \end{verbatim}
\end{breakbox}

大体ではあるが、断面積 は $7.0$, $a$ は 0.7 とした。断面積が
$7.0$ であるので、円筒だとするならば半径が $1.5$ 位の 瓶であると
考えられる。
