\begin{abstract}

授業中出された練習問題を、部分積分、置換積分を使って解く。

\section*{レポートの目的}
\addcontentsline{toc}{section}{レポートの目的}
前回まで勉強してきた微分積分を、実際に公式を使って計算する。
この練習問題で、積分の計算過程での式の変形に慣れる。

\section{今回使う基本的な公式}
%------------------------------------------------------
\subsection{部分積分}
積分の式を計算するために、掛け算の微分の公式を使って積分の公式
の形に変形してゆく手法。\\
微分の掛け算の公式は
\begin{eqnarray*}
(f(x)g(x))' = f'(x)g(x) + f(x)g'(x)
\end{eqnarray*}
である。この式の両辺を x で微分すると、
\begin{eqnarray*}
\int (f(x)g(x))'dx = \int f'(x)g(x)dx + \int f(x)g'(x)dx
\end{eqnarray*}
微分したものを積分すると、もとの関数になるので
\begin{eqnarray*}
f(x)g(x) = \int f'(x)g(x)dx + \int f(x)g'(x)dx
\end{eqnarray*}
これを移行して
\begin{eqnarray*}
\int f'(x)g(x)dx = f(x)g(x)- \int f(x)g'(x)dx 
\end{eqnarray*}
$\int x \cos(x) dx$ などの式の形に近いのでこの公式を
使って計算することが出来る。

\subsection{置換積分}
置換積分では、微分の公式である合成関数の公式を使って
積分の公式で計算できる形に変形させる。$\int f(x)dx$ を
$F(t)dt$ のような形にして解く。

合成関数の公式は
\begin{eqnarray*}
(F(x(t)))' = F'(x(t)) x'(t) 
\end{eqnarray*}
これに両辺を t で積分したものが公式である。
\begin{eqnarray*}
\int (F(x)) dx = \int F'(x(t)) x'(t) dt
\end{eqnarray*}

%-------------------------------------------
\section{部分積分}
%-------------------------------------------
\subsection{$\int e^x \cos x dx$}
まず、公式にあてはめる為に $f(x)=\cos x$, $g(x)= e^x$ とおく。
また、$e^x$ を積分したものは$e^x$, $\cos(x)$ を積分したものは
$\sin(x)$である。

\begin{eqnarray*}
\int e^x \cos x dx = e^x \sin x - \int e^x -\sin x dx  
\end{eqnarray*}
ここで、$\int e^x -\sin x dx$ をさらに部分積分する。
$f(x)= -\sin x$, $g(x)=e^x$ とおいて、
\begin{eqnarray*}
\int e^x -\sin x dx &=& e^x (-\cos x) - \int e^x(-\cos x) dx 
\end{eqnarray*}

もとの式に戻って
\begin{eqnarray*}
\int e^x \cos x dx = e^x \sin x - (e^x (-\cos x) + \int e^x \cos x dx) \\
\int e^x \cos x dx = e^x \sin x + e^x \cos x - \int e^x \cos x dx 
\end{eqnarray*}
ここで 右辺と左辺の $\int e^x \cos x dx$ が同じなので、右辺の式を移項すると
\begin{eqnarray*}
2\int e^x \cos x dx = e^x \sin x + e^x \cos x
\end{eqnarray*}
左辺の 2 をとって
\begin{eqnarray*}
\int e^x \cos x dx = \frac{1}{2} e^x \sin x + e^x \cos x
\end{eqnarray*}
右辺の$e^x$をくくって
\begin{eqnarray*}
\int e^x \cos x dx = \frac{1}{2} e^x(\sin x + \cos x)
\end{eqnarray*}
よってこの式は
\begin{eqnarray*}
\int e^x \cos x dx = \frac{1}{2} e^x(\sin x + \cos x)
\end{eqnarray*}
となる。

%----------------------------------------------------------
\subsection{$\int x^2 e^x dx$}
まず、公式にあてはめる為に $f(x)= e^x $, $g(x)= x^2$ とおく。
これを当てはめて
\begin{eqnarray*}
\int x^2 e^x dx = e^x x^2 - \int e^x 2x dx
\end{eqnarray*}
ここで k が定数の時、$\int k f(x)dx = k \int f(x)dx$ だから
\begin{eqnarray*}
\int x^2 e^x dx = e^x x^2 - 2\int e^x x dx
\end{eqnarray*}
となる。さらに $\int e^x x dx $ を部分積分して、
$f(x) = e^x$, $g(x) = x$ とおくと 
\begin{eqnarray*}
\int e^x x dx = e^x x - \int e^x dx \\
\int e^x x dx = e^x x - e^x 
\end{eqnarray*}
これを先程の式に戻して
\begin{eqnarray*}
\int x^2 e^x dx = e^x x^2 - 2(e^x x - e^x)
\end{eqnarray*}
$e^x$ でくくって
\begin{eqnarray*}
\int x^2 e^x dx = e^x (x^2 - 2x + 2)
\end{eqnarray*}

よってこの式は
\begin{eqnarray*}
\int x^2 e^x dx = e^x (x^2 - 2x + 2)
\end{eqnarray*}
となる。

%-------------------------------------------
\section{置換積分}
%-------------------------------------------
\subsection{$\int(2x+1)^{15} dx$}
公式に当てはめるために $t = 2x+1$ とする。
さらに、関数 $F(x) = x^{15}$ とする。
\begin{eqnarray*}
\int (F(x)) dx = \int (t)^{15} dt \\
\int (F(x)) dx = \int F'(t)・x'(t) dt 
\end{eqnarray*}
この式を使って、$dx$ と $dt$ の関係を調べる。
\begin{eqnarray*}
\frac{dt}{dx} = \frac{d(2x+1)}{dx}\\
\frac{dt}{dx} = 2\\
dx = \frac{1}{2}dt
\end{eqnarray*}
この$dx$を、問題の式にあてはめると
\begin{eqnarray*}
\int t^{15} \frac{1}{2} dt  \\
\frac{1}{2}\int t^{15} dt 
\end{eqnarray*}
積分して
\begin{eqnarray*}
\frac{1}{2}・\frac{t^{16}}{16}  \\
frac{1}{32} t^{16} 
\end{eqnarray*}
$t$ を元に戻して
\begin{eqnarray*}
frac{1}{32} (2x+1)^{16} 
\end{eqnarray*}

よって、この式は
\begin{eqnarray*}
frac{1}{32} (2x+1)^{16} 
\end{eqnarray*}
となる。

%-----------------------------------------------------
\subsection{$\int \cos(3x+1) dx$}
公式に当てはめる為に、$t=3x+1$ とする。
公式
\begin{eqnarray*}
\int (F(x)) dx = \int F'(t)・x'(t) dt 
\end{eqnarray*}
この $x'(t)$ について解けば、$dx$と$dt$ の関係を得ることが
出来るから、
\begin{eqnarray*}
\frac{dt}{dx} = \frac{d(3x+1)}{dx}\\
\frac{dt}{dx} = 3 \\
dx = \frac{1}{3}dt
\end{eqnarray*}
これを
\begin{eqnarray*}
\int cos(t) dt 
\end{eqnarray*}
に当てはめると
\begin{eqnarray*}
\int cos(t) \frac{1}{3}dt \\
\frac{1}{3} \int cos(t)dt \\
\end{eqnarray*}
積分して
\begin{eqnarray*}
\frac{1}{3} sin(t)
\end{eqnarray*}
t を元に戻して
\begin{eqnarray*}
\frac{1}{3} sin(3x+1)
\end{eqnarray*}
よってこの式は
\begin{eqnarray*}
\frac{1}{3} sin(3x+1)
\end{eqnarray*}
となる。

\section{追加の問題}
%-------------------------------------------
\subsection{$\int x (\log x)^2 dx$}
まず、公式に当てはめるために $f'(x) = x$, $g(x)= (\log x)^2$
とする。また、$f(x)=\frac{x^2}{2}$となる。
この式を部分積分して
\begin{eqnarray}
\int x(\log x)^2 dx = \frac{x^2}{2}(\log x)^2 - \int \frac{x^2}{2}((\log x)^2)' dx
\end{eqnarray}
また、$((\log x)^2)'$ は合成関数であるから、$F(x)=x^2$ とおいてこれを解く。
\begin{eqnarray*}
((\log x)^2)'= F'(\log x) \log 'x \\
((\log x)^2)'= 2\log x \frac{1}{x} \\
((\log x)^2)'= \frac{2\log x}{x} 
\end{eqnarray*}

そして
$\int \frac{x^2}{2}((\log x)^2)' dx$
を解くと、
\begin{eqnarray*}
\int \frac{x^2}{2}((\log x)^2)' dx= \int x \log x dx
\end{eqnarray*}

さらに、$\int x \log x dx$ を部分積分すると
$f'(x)=x$, $g(x)=\log x$ とおいて
\begin{eqnarray*}
\int x \log x dx = \frac{x^2}{2} \log x - \int \frac{x^2}{2} \frac{1}{x} dx \\
\int x \log x dx = \frac{x^2}{2} \log x - \int \frac{x}{2} dx \\
\int x \log x dx = \frac{x^2}{2} \log x - \int \frac{1}{2}x dx \\
\int x \log x dx = \frac{x^2}{2} \log x - \frac {1}{2} \int x dx 
\end{eqnarray*}
$\int x dx$ を積分して
\begin{eqnarray*}
\int x dx = \frac{x^2}{2}
\end{eqnarray*}
これを戻して
\begin{eqnarray*}
\int x \log x dx = \frac{x^2}{2} \log x - \frac {1}{2}・\frac{x^2}{2} \\
\int x \log x dx = \frac{x^2}{2} \log x - \frac {x^2}{4}
\end{eqnarray*}
この式が
\begin{eqnarray*}
\int \frac{x^2}{2}((\log x)^2)' dx = \frac{x^2}{2} \log x - \frac{x^2}{4}
\end{eqnarray*}
となる。
これを 1 の式に当てはめると
\begin{eqnarray}
\int x(\log x)^2 dx = \frac{x^2}{2}(\log x)^2 - (\frac{x^2}{2}\log x - \frac{x^2}{4}) \\
\int x(\log x)^2 dx = \frac{x^2}{2}(\log x)^2 - \frac{x^2}{2}\log x + \frac{x^2}{4} 
\end{eqnarray}
$\frac{x^2}{4}$で全体をくくって
\begin{eqnarray}
\int x(\log x)^2 dx =  \frac{x^2}{4}( 2 (\log x)^2 - 2\log x + 1)
\end{eqnarray}
よって、この式は
\begin{eqnarray}
\int x(\log x)^2 dx =  \frac{x^2}{4}( 2 (\log x)^2 - 2\log x + 1)
\end{eqnarray}
となる。
